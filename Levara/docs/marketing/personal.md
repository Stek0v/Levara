# LevaraOS Personal — локальная память для вашего ИИ

**Ваш ИИ-помощник наконец-то перестанет всё забывать. И всё останется только у
вас.**

## Для кого

Один разработчик или энтузиаст, который ежедневно работает с ИИ-ассистентами —
Claude Code, Cursor, Codex и подобными — на своей машине.

## Боль

- Каждая новая сессия — с чистого листа: заново объясняешь проект, стек,
  договорённости.
- Полезные находки и решения теряются, как только закрыл окно.
- Не хочется отдавать наработки в чужое облако.

## Что даёт Personal

Локальная долговременная память для ваших ИИ-агентов. Запускается на ноутбуке,
хранит всё у вас, работает без сложной инфраструктуры.

| Возможность | Что это даёт вам |
|---|---|
| Локальное хранилище (SQLite + файлы) | Никаких внешних серверов и облаков — данные на вашей машине |
| Поиск по смыслу + по ключевым словам | Находит нужное, даже если формулировка другая |
| «Дворец памяти» (memory palace) | Помощник помнит факты, решения и предпочтения между сессиями |
| Рабочее пространство в Markdown | Ваши заметки — источник правды, ИИ читает и дополняет их |
| Авторизация опциональна | На локальной машине можно вообще без паролей — запустил и работаешь |

## Почему сейчас

ИИ-ассистенты стали достаточно сильными, чтобы писать код, проводить ревью и
планировать задачи, но их память всё ещё часто живёт в одном чате. Personal
закрывает этот разрыв: агент получает локальный, индексируемый контекст проекта
без SaaS-зависимости.

## Сценарий

Утром открываете Cursor. Спрашиваете: «На чём мы остановились вчера по модулю
оплаты?» — и ассистент, благодаря LevaraOS, сразу поднимает контекст: какое
решение приняли, что отложили, какие были подводные камни. Без пересказа.

## Доказательства в продукте

- `LEVARA_PROFILE=personal` не требует обязательной авторизации.
- SQLite и локальное файловое хранилище подходят для одиночного разработчика.
- MCP tools дают агенту `save_memory`, `recall_memory`, `wake_up`,
  `workspace_read`, `workspace_write` и `workspace_search`.
- Markdown workspace оставляет файлы читаемыми человеком даже без Levara.

## Почему LevaraOS

- **Приватность по умолчанию** — данные не покидают компьютер.
- **Быстро** — поиск в среднем около 2,6 мс, ощущается мгновенным.
- **Просто** — один файл-настройка, одна команда запуска.
- **Без потолка** — когда появится второе устройство или команда, просто включите
  следующий уровень. Переезжать на другой продукт не нужно.

## Как начать

```bash
cp deploy/profiles/personal.local.env.example .env
./levara-server -config-check   # проверить, что конфигурация корректна
./levara-server                 # запустить
```

Дальше — подключаете любимого ИИ-агента через MCP, и память начинает работать.
